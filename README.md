# Prototyping App
https://www.figma.com/design/0rlckFJzYI46JGz5e1iAo7/Mobile-Apps-–-Prototyping-Kit--Community-?node-id=101-306&p=f&t=vxQSFeOJcilgspEs-0

## Функции
- Онбординг с выбором интересов
- Подписка
- Авторизация (логин/регистрация + код подтверждения)
- Главный экран с проектами
- Explore с поиском шаблонов
- Готовые шаблоны: E-commerce, Events (с картой)
- Корзина, оплата, фидбек

## Технологии
- Flutter + Dart
- go_router
- flutter_bloc / Cubit
- Dio
- Hive
- Clean Architecture
